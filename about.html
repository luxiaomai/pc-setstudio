創立理念
SET STUDIO于2017年成立，主要工作領域为軟裝設計，傢俱定制，品牌升級，包括私人住宅和商業空間項目。
SET STUDIO，SET分別解釋為Simply（簡約）、Esthetic（審美）、Timeless（永恆）。主創設計師曾于英國藝術學院深造，受北歐設計的啟發，將簡約而永恆的美學帶入市场，有别于现有的千篇一律的設計格局，简约而不简单；并期望将内敛而具艺术感的設計風格，高質量而舒適的傢俱分享給對生活品味有相同追求的人。
设计其实和生活一样，都在放纵和克制，简单和複雜之間选择与徘徊，只有在其中找到最完美的平衡點，在沒有任何元素可以增加或者減少的時候，才能真正感受生活之美。
